Bac à sable

Apprenez à votre Buggle à faire ses premiers pas ou bien essayez-vous aux trajectoires les plus complexes. Ici, rien ne vous arrêtera !

Les commandes à votre disposition sont : 

avance()[!java|c];[/!]
recule()[!java|c];[/!]
droite()[!java|c];[/!]
gauche()[!java|c];[/!]
baisseBrosse()[!java|c];[/!]
leveBrosse()[!java|c];[/!]
setCouleurBrosse()[!java|c];[/!]
[!java|c]

N'oubliez pas le ; final, car cela indique au compilateur que l'instruction est terminée.

[/!] [!java|c]

Les couleurs à votre disposition sont : Color.RED, Color.GREEN, Color.BLUE, Color.YELLOW, Color.BLACK, Color.WHITE.

Par exemple, pour modifier la couleur de la brosse en rouge il faut écrire : setCouleurBrosse(Color.RED);

Vous pouvez également demander à votre Buggle d'effectuer plusieurs pas dans la même direction. Pour avancer de deux pas,vous pouvez écrire : avance(2);

[/!]

Toutes les commandes à votre disposition sont rappelées dans l'onglet API.

Une fois terminé, cliquez sur le bouton se trouvant dans la section Cas de tests pour lancer votre programme. Quand cela fonctionne, vous pouvez passer à la séquence suivante. Pour changer de séquence, cliquez sur le bouton se trouvant en haut à gauche de votre écran pour faire apparaître le menu latéral. Sélectionnez ensuite le prochain élément de la liste de séquences.
